# Редиректы со старых страниц раздела LADA_Granta

Скомплектовано страниц: 31 мягких редиректов + 3 страниц 404
(шаблоны повторяют раздел GAZelle Next).

## Куда выкладывать

Скопируйте ВСЕ файлы из этой папки в каталог СТАРОГО раздела сайта
(`/LADA_Granta/`) поверх существующих одноименных страниц.

Механика редирект-страниц (как в GAZelle Next):
- сохранены старые title/description/keywords;
- `canonical` на новую SEO-страницу + `noindex, follow` (Google и Яндекс
  признают страницу переехавшей и передадут ссылочный вес новому URL);
- посетитель-человек перебрасывается скриптом через 10 секунд на новую
  страницу (поисковый робот на странице остается).
